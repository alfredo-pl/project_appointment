# Getting Started 
To install the project locally, you must first verify that it contains the following versions
## Prerequisites
`$ Rails 6`
`$ Ruby 3`
`$ yarn 1.22`
`$ npm 6.14`
`$ node v10`
## Installation
1.Clone repository

``$ git clone https://github.com/alfredo-pl/project_appointment.git``

2.Bundle install

  `$ bundle`

3.yarn install

  `$ yarn`

4.Create db

  `$ Rails db:create`

5.Run Migrations

  `$ Rails db:migrate`

6.Run seeds

  `$ Rails db:seed`

### Ready!!
